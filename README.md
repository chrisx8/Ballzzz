## Run the game

**All files listed in this section are REQUIRED to run the game.**

- [Download ZIP](https://github.com/chrisx8/Ballzzz/archive/master.zip)
- Install dependencies using `pip`:

  ```bash
  pip install -r requirements.txt
  ```

- Run `ballzzz.py`
